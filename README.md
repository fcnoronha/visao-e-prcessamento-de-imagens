# Visão e processamento de imagens

Repositório para os projetos da disciplina Visão e processamento de imagens.

Instale as dependências necessárias usando o comando

```bash
pip3 install -r requirements.txt
```

## Como executar o notebook `ep01.ipynb`

- Execute o primeiro bloco da seção 2 para importar a base de imagens do nosso Github
- Agora basta executar as seções 3 e 4 para gerar as tabelas sumarias de dados e a visualização de algumas imagens da base

## Como executar o notebook `ep03.ipynb`

- Clone este repositório para baixar as bases dados.
- Agora basta rodar as células de código na ordem que elas aparecem
